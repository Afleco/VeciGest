# Bienvenido a VeciGest

VeciGest es una aplicación de gestión vecinal para facilitar la vida a las comunidades de vecinos, permitiendo el manejo de cuotas, incidencias y notificaciones entre muchas otras funcionalidades, todo a través de una sola aplicación.

## Librerías de React Native + Expo necesarias para su ejecución:

**SUPABASE** \
npx expo install @supabase/supabase-js @react-native-async-storage/async-storage react-native-paper react-native-safe-area-context -- --legacy-peer-deps

**Navegacion con menu desplegable** \
npx expo install @react-navigation/native @react-navigation/drawer react-native-gesture-handler react-native-reanimated
